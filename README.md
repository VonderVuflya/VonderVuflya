## 🚀 What I'm building

### 🌳 [Yggdrasil](https://github.com/VonderVuflya/Yggdrasil) — durable memory for AI coding agents

[![PyPI](https://img.shields.io/pypi/v/yggdrasil-memory?label=PyPI&color=3775A9&logo=pypi&logoColor=white)](https://pypi.org/project/yggdrasil-memory/)
[![Stars](https://img.shields.io/github/stars/VonderVuflya/Yggdrasil?style=flat&color=eac54f&logo=github)](https://github.com/VonderVuflya/Yggdrasil/stargazers)
[![License: Elastic 2.0](https://img.shields.io/badge/license-Elastic%202.0-2b7489)](https://github.com/VonderVuflya/Yggdrasil)

One shared, persistent brain that **Claude Code, Codex, and any MCP host** plug into — it remembers your decisions, lessons and project status across sessions, tools and projects.

- ⚡ **Zero-dependency** — pure Python + SQLite/FTS5. No Node, no vector DB, no cloud.
- 🔒 **100% local & private** — your memory never leaves your machine; you own the data.
- 🧠 Curated & **semantically de-duped**, with optional local models (Ollama) for semantic search.
- 📦 Shipped to **PyPI · npm · Homebrew · the MCP Registry** — install-and-go.

### 🛰️ [Pullsar.space](https://pullsar.space) — every job, one feed

My own product: a **job board** that pulls fresh vacancies straight from **Telegram channels and HH.ru** into one clean feed. TypeScript end-to-end, architected with **Feature-Sliced Design**, self-hosted on Coolify.
